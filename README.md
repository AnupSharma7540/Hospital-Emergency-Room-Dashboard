# 🏥 Hospital Emergency Room Dashboard

An interactive Microsoft Excel dashboard designed to analyze Hospital Emergency Room (ER) performance using KPIs, Pivot Tables, Pivot Charts, Slicers, and Data Visualization techniques.

---

## 📌 Project Overview

This project provides an interactive dashboard to monitor and analyze Emergency Room performance. It helps identify patient trends, waiting times, admission status, referrals, and overall hospital efficiency through dynamic visualizations.

---

## 🎯 Objectives

- Analyze Emergency Room performance
- Monitor patient waiting time
- Track patient admissions and discharges
- Identify peak patient traffic
- Improve operational decision-making through data
